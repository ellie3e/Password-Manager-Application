# Password-Manager-Application
Password manager is a python application for securely storing and managing your passwords. It uses PyQt5 for the graphical user interface and SQL for database storage. Passwords are encrypted using the Fernet encryption library, ensuring the security of your sensitive data.

## Features

- Store and manage passwords for various services.
- Data is stored in an SQL database for easy retrieval.
- Passwords are encrypted with fernet for added security.
- User-friendly graphical user interface (GUI).

## Installation

Before using password manager, make sure you have the following dependencies installed:

- Python 3.x
- PyQt5
- cryptography
